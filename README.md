# RCSprint

A 1/10-scale **dirt-oval RC sprint car** racing game built in the browser with **Babylon.js**, modeled on the real **Team Losi 22S Sprint** (TLR 22 platform).

- Driver-stand camera (trackside RC vantage)
- Sim-leaning physics (raycast vehicle, slip-based tires, evolving dirt grip, tire wear, car setup)
- Career/championship across 15 progressively harder dirt ovals
- Full ~8–10-car fields, winged sprint cars
- Gamepad + keyboard control

## Run

```
npm install
npm run dev
```

Then open http://127.0.0.1:5173
